CRIAÇÃO DE UM CHATBOT

O que foi proposto?

Foi proposto o desenvolvimento de um ChatBot com o objetivo de facilitar a pesquisa personalizada, esclarecer dúvidas e aprimorar a integração com nossa ferramenta (uma extensão de navegador para pesquisa).

Abordagem

1- Chatbot Retrieval

Esse tipo de chatbot recupera respostas predefinidas de um conjunto de dados existente com base nas entradas dos usuários, em vez de gerar respostas totalmente novas por meio da geração de linguagem natural.


1.1 - Modelo de Chatbot com NLTK

Uma abordagem mais simples em relação ao chatbot foi utilizando a biblioteca NLTK. Nessa abordagem, era necessário o input preciso do usuário para que fosse retornado a resposta correspondente ao exato input. Além disso, foi desenvolvido o começo do fornecimento de artigos de wikipedia.

Disponível em: Link para o Google Colab


1.2 - Modelo Bag Of words

Ele utiliza representações de BoW para realizar inferências e determinar a melhor resposta, dentre as opções previamente anotadas, a ser fornecida ao usuário. Nossa intenção ao treinar este modelo era melhorar a compreensão pelo chatbot das frases fornecidas pelo usuário, mesmo que elas não correspondessem exatamente às frases anotadas.

Realizamos treinamentos ao longo de 1000 épocas e conduzimos diversos experimentos com seus hiperparâmetros, no entanto, a abordagem ainda exigia entradas muito precisas pelo usuário para que o chatbot pudesse compreender adequadamente. Ficou determinado, como uma possível melhora dessa abordagem, o uso de representações densas das palavras para aprimorar o contexto do input fornecido.

Disponível em: Link para o Github


2- Chatbot Generativo

Um chatbot generativo é um tipo de chatbot que é capaz de gerar respostas de forma autônoma, criando texto novo e contextualmente relevante em vez de apenas selecionar respostas predefinidas de um conjunto de dados. Esse tipo de chatbot utiliza LMMs como o GPT3 para fornecer respostas mais dinâmicas.


2.1 - Modelo através da API do ChatGPT

Por fim, essa abordagem utiliza diretamente a API do ChatGPT para adquirirmos as respostas a partir dela, e usamos o Gradio para obtermos uma interface.
