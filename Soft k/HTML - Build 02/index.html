IDEAÇÃO PARA BUILD 2

O SOFT K 4.0 tem a finalidade de desenvolver um Produto Mínimo Viável (MVP) destinado a apoiar profissionais de Recursos Humanos no processo de seleção de novos colaboradores.

Nossa meta é reduzir os encargos associados às más contratações. Neste contexto, no presente card BUILD, pretendemos trabalhar com os três datasets abordados na instruction anterior, assim como evoluir da fase de exploração para conectarmos com o mundo.

Realizamos uma análise eliminatória dos datasets da “ SOFT K 4.0 - INSTRUCTION 3 - Datasets adequados para a solução” que consistia na exploração dos Datasets disponíveis, construindo um guia informativo sobre cada um deles, e estes foram os que atenderam aos nossos critérios:

O objetivo foi:

  • Realização de uma análise exploratória dos dados;
  • Execução do pré-processamento;
  • Implementação de um modelo de visão computacional para o conjunto de dados em vídeo e aplicação de um modelo de Processamento de Linguagem Natural (NLP) para os conjuntos de dados textuais.
  • Introdução prática ao Streamlit para integração subsequente dos modelos após treinamento;
  • Estudo de mercado e conexão com stakeholders para o desenvolvimento do SOFT K 4.0 como produto.
